# Отчёт о тестировании Precision

## Краткое описание

13.08.2021 было проведено функциональное тестирование приложения Precision.

На тестирование затрачено: 1 час

В результате тестирования выявлены следующие дефекты:

Полученное в результате число имеет много цифр после запятой.

## Описание процесса тестирования

В качестве тестовых данных использовались данные из домашнего задания:

Переменная 0.3 и переменная 0.6. 
Ожидаемый результат: сложение переменных в результате превращается в дробь с одной цифрой после запятой.
Фактический результат: при сложении переменных после запятой множество цифр.

Тестирование производилось в следующем окружении:
* Windows 10 x64
* Java 11
